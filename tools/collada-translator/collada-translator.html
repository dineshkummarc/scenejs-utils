Enter the location of a .DAE file then click Translate to convert it into a SceneJS JSON subgraph, nicely packaged in a SceneJS Module for easy integration into your scene graph.


Your new module will have the same image URLs as the original COLLADA file, so it needs to be at the same location relative to the images.

With your module file saved where your scene file can get it, you can then import it like so:


                    SceneJS.requireModule("http://foo.com/seymourplane_triangulate.js");

The module will organise its <visual_scene> elements and <camera> views as Nodes within libraries, which you may instantiate within your scene graph. The available Symbol IDs will be listed in the function's comments.

For example, to view the Seymour Plane example's <visual_scene> "VisualScene" through one of it's cameras:


               SceneJS.instance({ target: "seymourplane_triangulate.camera1" });
